# wcr command

wcr command by rust

## Install

```shell
cargo install --path . --force
```
## Uninstall

```shell
cargo uninstall wcr
```

## Usage

Usage: wcr [OPTIONS] [FILE]...

Arguments:
[FILE]...  [default: -]

Options:
- -l, --lines  
- -w, --words  
- -m, --chars  
- -c, --bytes  
- -h, --help   Print help
